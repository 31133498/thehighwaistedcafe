# **App Name**: High Waisted Cafe Online

## Core Features:

- Header Section: Includes company logo placeholder, navigation links (Home, Menu, About Us, Contact), and an Order Now button.
- Hero Slider Section: Features a full-width image slider with high-resolution images and welcoming text overlay with a call-to-action button.
- Editable Menu Section: Displays café items with images, names, short descriptions, and prices, with editable fields for descriptions and prices. Includes an Add to Cart button.
- About Us Section: A short, warm description of the café emphasizing quality, comfort, and hospitality.
- Contact Section: Includes a simple contact form, a WhatsApp chat link button, and clearly displayed café contact details.
- Footer Section: Features social media icons and copyright text.

## Style Guidelines:

- Primary color: Coffee Brown (#A67B5B) to evoke warmth and comfort.
- Background color: Cream (#F5F5DC) provides a soft and inviting backdrop.
- Accent color: Gold (#D4AF37) adds a touch of elegance and highlights important elements.
- Body font: 'Poppins', a geometric sans-serif for a readable, contemporary feel.
- Headline font: 'Playfair', a modern serif similar to Didot, for headlines or small amounts of text; Note: currently only Google Fonts are supported.
- One-page layout with smooth scrolling between sections for easy navigation.
- Subtle parallax or soft textured background related to coffee for a stunning visual effect.
